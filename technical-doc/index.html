CSS

My first thought was to use flexbox. Just 2 boxes, one for the navbar on the left and the main section (on the right, thanks). But the challenge asks for the navbar always visible and on the left. So just some simple position: fixed and 0 from top and left should do it.

Here's the code:

                
                    #navbar {
                        width: 300px;
                        position: fixed;
                        left: 0;
                        top: 0;
                        height: 100%;
                    }

                    main {
                        margin-left: 300px;
                    }

The Media Query

The media query is pretty simple. Both boxes get the whole width. The navbar is reset to its initial position. A max height is set to keep the scrollbar.

@media (max-width: 700px) { #navbar { position:initial; width: 100%; height: 150px; } #navbar ul { height: 100px; } main { margin-left:0; margin-top: 150px; } }

pre + code

The <code> tag is used t include code snippets in the html document. However, whitespace is removed, and line breaks won't show.

We want the browser to show line breaks and white spaces. The <pre> tag defines preformatted text, and will keep spaces and line breaks. We just wrap the code inside <pre> tags.

The media query code above is wrapped in <code> tags only. Here's the same code block above, now with the pre + code tags:


                    @media (max-width: 700px) { 
                        #navbar {
                            position:initial;
                            width: 100%;
                            height: 150px;
                        }
                        #navbar ul {
                            height: 100px;
                        }
                        main {
                            margin-left:0;
                            margin-top: 150px;
                        }
                    }

More readable!
